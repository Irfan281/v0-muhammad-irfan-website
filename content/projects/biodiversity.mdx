# Biodiversity Information System

A comprehensive Android application for biodiversity information management.

## Overview

Designed and developed for the Department of Biology at Syiah Kuala University, this system provides a mobile interface for accessing and managing biodiversity data. The project included both Android frontend development and Express.js backend implementation.

## Features

- **Species Database**: Comprehensive information on various species
- **Search and Filter**: Find species by various criteria
- **Offline Access**: Access data without internet connection
- **Admin Dashboard**: Backend interface for data management
- **Data Optimization**: Cleaned and optimized existing species data

## Technical Details

- **Architecture**: MVVM with Clean Architecture principles
- **Dependency Injection**: Dagger-Hilt
- **Reactive Programming**: Kotlin Flow
- **Local Storage**: Room Database for offline-first approach
- **Backend**: Express.js with MongoDB
- **API Communication**: Retrofit with Kotlin Serialization

## Achievements

- Optimized existing species data by 40% by cleaning up duplicate and unclear entries
- Implemented a robust offline-first approach that ensures functionality without internet
- Created an intuitive admin interface for university staff to manage data
- Designed a scalable architecture that can accommodate future expansion of the database
